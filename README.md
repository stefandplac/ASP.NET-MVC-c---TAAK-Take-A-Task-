# ASP.NET MVC c# -TAAK-Take-A-Task- web application  using C#,	SQL Server, Linq, freelancing for home based activities like painting interior or exterior of buildings, furniture assembly or any other similar activity where users can register and post a task or many tasks(small jobs) and other users can make an offer for task execution.
